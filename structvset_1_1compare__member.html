template<typename S, typename M, M S::* m, typename C = std::less<M>>

Позволяет создавать компаратор отдельного поля структуры или класса

S- тип структуры
M- тип поля структуры
m- указатель на поле структуры
C- копаратор для M (по умолчанию std::less<M> )
