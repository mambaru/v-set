v-set
v-set Документация

Библиотека для построения контейнеров для хранения объектов фиксированного размера

  • Документация doxygen.
  • Репозитарий на github.com.
  • Отчет coverage.

Сборка и установка

Это header-only библиотека, компиляция не требуеться. Зависит от faslib Для компиляции примеров и тестов достаточно:

git clone https://github.com/mambaru/v-set.git
cd v-set
make

Если установлен doxygen то в папке /doc/html будет документация.
